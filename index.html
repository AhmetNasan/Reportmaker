Powerful Features

Everything you need to manage engineering contracts and projects efficiently

Contract Management

Create, edit, and track engineering contracts with full client metadata and comprehensive contract lifecycle management.

Project Tracking

Connect projects to contracts with real-time progress monitoring and status updates across all phases.

AI Inspections

Quick Setup

Get up and running in minutes

Setup Instructions

1

Clone and Install

git clone <repository-url>
cd engineering-platform
npm install
2

Configure Environment

cp .env.example .env
# Add your DATABASE_URL, PORT, NODE_ENV
3

Database Setup

npm run db:push
4

Start Development

npm run dev

Database Schema

Optimized relational database design

contracts

  • • id (Primary Key)
  • • contract_number
  • • client_name
  • • status, dates, metadata

projects

  • • id (Primary Key)
  • • contract_id (Foreign Key)
  • • name, progress
  • • coordinates, status

contract_files

  • • id (Primary Key)
  • • contract_id (Foreign Key)
  • • file_path, file_type
  • • upload_date, size

inspections

  • • id (Primary Key)
  • • project_id (Foreign Key)
  • • image_path
  • • defects_found, analysis
