## 2025-05-22 - Improved Work Order Table Interactivity
**Learning:** In data-heavy applications, visual synchronization between map views and table rows is critical for user orientation. Selection states should be reflected both visually (highlight) and semantically (ARIA).
**Action:** Implement combined selection highlighting and ARIA attributes for table rows, and ensure the table syncs when an asset is highlighted on the map.
