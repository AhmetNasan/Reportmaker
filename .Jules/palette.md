## 2026-06-07 - [Enhanced Selection and Focus Indicators]
**Learning:** Consistently using 'aria-selected' along with visual CSS classes (like '.selected') improves accessibility for screen readers while maintaining a clear visual state for users. Enhancing focus indicators (e.g., adding box-shadow to '.filter-input:focus') ensures keyboard navigation is intuitive and accessible.
**Action:** Always pair visual state changes in tables with appropriate ARIA attributes and ensure focus states are clearly visible beyond just border-color changes.
